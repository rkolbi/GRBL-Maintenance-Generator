Help & Safety Overview

Use this generator to create safe, spindle-off G-code for CNC maintenance routines such as grease distribution, axis cycling, and positioning for manual lubrication. All routines use conservative feeds and G1-only motion to minimize mechanical stress.

Why This Routine Matters

Simply moving an axis end-to-end only pushes grease around. This program uses a randomized “pecking” or “shingling” motion—moving forward, then partially back—to work lubricant into the bearings and distribute it evenly along the rails. This ensures smooth motion, prevents wear, and extends the life of your machine’s mechanical components.

The Truth About Woodworking CNCs

Community discussions reveal three universal truths about CNC maintenance:

  1. The Biggest Problem: Abrasive Paste
    Grease attracts dust. Fine sawdust—especially from MDF—mixes with exposed grease on the rails, forming a gritty, abrasive “lapping compound.” As the machine moves, this paste gets forced into bearing block seals, rapidly wearing down the internal ball bearings.
  2. The Best Solution: Wipe Dry
    Rails should never be left “wet” with grease. Lubricant belongs *inside* the bearing blocks—ideally via zerk fittings—not *on* the rails. The proper routine is: grease, jog the axes, then wipe the rails completely clean and dry.
  3. The Hardest Hurdle: Tedium
    Maintenance often fails due to inconsistency. Operators typically “grease and jog,” moving the machine only a few times. This does not purge old grease and is tedious, so the new grease rarely spreads evenly.

Ideal Greasing Workflow

Grease is a dust magnet—rails that are “wet” with it quickly become abrasive. Follow this workflow to prevent damage:

  1. Home the Machine: Run your machine's standard homing cycle ($H) first.
  2. Apply Lubricant: Apply grease with a zerk fitting on the bearing block. If not available, use a *very thin* bead along the rail grooves.
  3. Run This Program: Load and run the G-code generated by this tool. The pecking motion pulls grease into the bearing blocks and distributes it internally.
  4. Wipe Rails Dry: After completion, use a clean, dry, lint-free cloth to remove *all* excess grease. Rails should appear slick and shiny, but never “wet” or goopy.

Goal: Grease should reside *inside* the bearings, not *on* the rails.

Pre‑Flight Safety Checklist

  • Ensure spindle is OFF and cannot auto-start (physical disconnect recommended).
  • Remove all tools/end mills from the spindle.
  • Clear the work area of clamps, fixtures, and materials.
  • Verify accurate machine travel limits (X/Y/Z).
  • Confirm machine is homed if required by your controller.
  • Check lubrication points and have grease/oil ready if using “service points” mode.
  • Simulate or preview the G-code before running.
  • Remain at the machine for the entire routine.

Settings Overview

1. Axis Limits (Absolute Coordinates)
  • X/Y/Z Start (Home): Machine's absolute coordinates after homing switches trigger (e.g., -10).
  • X/Y/Z End (Far): Maximum travel from the home position (e.g., -1270).
2. Routine Selection
  • Select Axes to Exercise: Choose which axes to run. Uncheck axes you do not want to exercise.
  • Complete Cycles: Number of times to run the full routine for selected axes.
  • Z-Axis Sub-Cycles: Number of up-and-down Z routines per complete cycle (recommended: 2–3).
3. Motion Settings
  • XY Min/Max Fwd Step (mm): Randomized forward peck distance for X/Y.
  • XY Rev Step (mm): Reverse peck distance for X/Y (one value is sufficient).
  • Z Min/Max Fwd Step (mm): Forward peck distance for Z (smaller than XY, e.g., 5–10mm).
  • Z Rev Step (mm): Reverse peck for Z (e.g., 2mm).
  • XY/Z Peck Speed (F): Work speed for pecking motion. Slower speeds ensure proper grease distribution.
  • Rapid Feed (F): Speed for initial Z lift and return home.
  • Exercise Feed (F) (NEW): The speed used for the X/Y moves in the "Generate Exercise G-Code" routine. This is separate from the pecking speed.
  • Pause Between Routines (sec): Dwell time between axis routines for inspection (e.g., G4 P5).
Speed Guidelines:
  • Work Speed: Slow is better; allows grease to penetrate bearings. Example: F1000 (XY), F500 (Z).
  • Travel Speed: Fast is fine; used only for initial/final Z lifts. Default F3000 is safe.
4. Positional Controls
  • Y-Pos for X-Cycling: Y-coordinate for X-axis back-and-forth routine (e.g., -1240).
  • X-Pos for Y-Cycling: X-coordinate for Y-axis routine (e.g., -10).
  • X/Y-Pos for Z-Cycling: X/Y location for Z-axis routine (e.g., X-10, Y-1240) for convenient access.
5. Actions (The Buttons)
  • Generate Greasing G-Code: This is the main function. It uses the "pecking" motion (Settings 3) at the specified locations (Settings 4) to perform a deep lubrication and purging routine.
  • Generate Exercise G-Code: This is a simpler, faster routine. It does *not* use pecking. It runs a full-speed (using "Exercise Feed") rectangular lap of the machine's X/Y limits (clockwise, then counter-clockwise) and cycles the Z-axis once per lap. This is good for a quick daily "warm-up" or for testing motion after assembly.
